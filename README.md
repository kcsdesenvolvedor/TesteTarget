
# Desafio Técnico - Starget Sistemas

Este repositório contém a resolução do desafio técnico proposto pela Starget Sistemas. O objetivo deste projeto é demonstrar habilidades em lógica de programação, manipulação de dados, modelagem de banco de dados e resolução de problemas analíticos.

## 💻 Sobre o Desafio

O teste consiste em 5 questões fundamentais que abrangem diferentes áreas do desenvolvimento de software:


- Lógica de Iteração: Análise de fluxo e controle de variáveis em laços de repetição.
- Raciocínio Lógico: Identificação de padrões e sequências numéricas.
- Algoritmos e Vetores: Processamento de dados de faturamento, cálculos estatísticos e otimização de performance.
- Banco de Dados: Modelagem lógica (Entidade-Relacionamento) e construção de consultas SQL complexas.
- Análise Cinemática e Lógica: Resolução de um problema clássico de encontro de objetos envolvendo variáveis de tempo e obstáculos (pedágios).
